## Visão geral
BuscaPeças HUB é a solução Android de alta performance para conectar clientes a CDVs parceiros reais.

- ✅ **APK Admin / lojista/vendedor**: acesso exclusivo para lojista/vendedor; cadastra, valida e organiza CDVs confiáveis.
- ✅ **APK Cliente**: encontra peças disponíveis e gera contato direto com o parceiro.

## Por que é relevante
Este projeto une duas experiências essenciais:
- gestão profissional de parceiros
- busca de peças com estoque real e transparência total

## Tecnologias usadas
- Android nativo
- Java 8
- Gradle
- Retrofit + Gson
- Firebase Messaging
- MySQL backend

---

## APK Admin / lojista/vendedor

### Propósito
O APK Admin / lojista/vendedor centraliza o cadastro e a validação dos CDVs parceiros, garantindo que apenas parceiros aprovados entrem na plataforma.

### Benefícios
- Cadastro completo de CDVs reais
- Validação de parceiros antes de aparecerem para o cliente
- Gestão de parcerias com segurança e controle
- Base confiável para alimentar o APK Cliente

---

## APK Cliente

### Propósito
O APK Cliente oferece uma experiência direta para quem busca peças: consulta o estoque, mostra o parceiro e facilita o contato imediato.

### Benefícios
- Busca por peça, código, descrição e região
- Exibição de CDVs com estoque confirmado
- Contato direto via telefone e WhatsApp
- Reserva de peças para agilizar a venda

---

## Público-alvo
- **APK Admin / lojista/vendedor**: donos de CDVs, gestores de parcerias e operadores de marketplace.
- **APK Cliente**: oficinas, mecânicos, desmontadoras e consumidores finais.

---

## Mensagem final
BuscaPeças HUB é a plataforma criada para profissionalizar o mercado de peças usadas, com controle de parceiros e resultados reais para o cliente final.
